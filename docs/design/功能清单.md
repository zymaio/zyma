# 智码（zyma）功能清单 - 进度更新 (v1.0 达成)

## 1️⃣ 核心内核功能 (✅ 已完成)

| 功能类别 | 功能点 | 状态 | 描述 |
| :--- | :--- | :--- | :--- |
| 文件操作 | 打开 / 保存 / 另存为 | ✅ | 支持本地文件读写、另存为对话框 |
| 编辑功能 | 多标签管理 (Tabs) | ✅ | 支持同时打开多个文件、脏检查提醒 |
| 编码支持 | UTF-8 / 自动探测 | ✅ | 支持 UTF-8，非 UTF-8 文件使用 Lossy 转换防止崩溃 |
| 界面显示 | 语法高亮 | ✅ | 集成 CodeMirror 6，支持 20+ 种主流语言 |
| Markdown | 编辑 + 实时预览 | ✅ | 左右分屏显示，GitHub Dark 风格渲染 |
| 搜索 | 全局搜索 | ✅ | 极速扫描全项目字符，点击跳转文件 |
| 主题 | 深色 / 浅色模式 | ✅ | 动态切换，包括编辑器主题同步切换 |
| 国际化 | 多语言 (i18n) | ✅ | 支持简体中文、繁体中文、英文 |
| 持久化 | 配置存储 | ✅ | 自动保存字号、主题、语言等偏好设置 |
| 窗口控制 | 状态恢复与防闪烁 | ✅ | 毫秒级无闪烁启动，自动恢复位置、大小及最大化状态 |
| 系统集成 | 右键菜单 | ✅ | Windows 下一键添加/移除“在智码中编辑” |
| 运行模式 | 单实例/多实例 | ✅ | 支持复用现有窗口或开启新窗口 |
