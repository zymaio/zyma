# Zyma 功能特性清单 (v1.1)

## 1. 高性能编辑器内核 (Core)
- [x] **极致启动**: Rust 并行初始化，毫秒级进入工作状态。
- [x] **20+ 语言高亮**: 动态加载，内存占用极低。
- [x] **Markdown 专家**: 实时预览，GitHub 风格，图片深度适配。
- [x] **全局搜索系统**: Rust `globset` + `ripgrep` 级扫描。

## 2. 工业级插件架构 (Platform)
- [x] **标准 API**: 对标 VS Code 的 `workspace`, `window`, `system` 命名空间。
- [x] **事件驱动系统**:
    - [x] 文件保存、创建、删除、修改全监听。
    - [x] 标签页切换、选区变化感知。
    - [x] 窗口焦点自动识别。
- [x] **生命周期管理**: 插件零干扰热加载与 `Disposable` 销毁机制。
- [x] **零耦合设计**: 插件物理级隔离，主程序源码纯净。

## 3. 智能输出系统 (Logging)
- [x] **幽灵式显隐**: 有输出即现，平时静默，持久驻留。
- [x] **监控模式**: 一键弹出独立窗口，支持跨显示器自由布局。
- [x] **坐标记忆**: 记录每一个频道窗口的绝对位置与大小。
- [x] **高级渲染**: 关键字染色、全选复制反馈、多语言支持。

## 4. 系统集成与体验 (System)
- [x] **路径标准化**: 引入全系统 `pathUtils` 工具，消灭路径冲突。
- [x] **窗口恢复**: 记住主窗口位置及最大化状态。
- [x] **双源更新**: GitHub/Gitee 双镜像全自动在线升级。
- [x] **原生增强**: Windows 右键菜单深度集成。
