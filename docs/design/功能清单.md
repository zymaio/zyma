# Zyma 功能特性清单

## 1. 核心编辑 (Core Editor)
- [x] **高性能内核**: 基于 Rust + CodeMirror 6，支持大文件极速打开。
- [x] **多语言高亮**: 内置 20+ 种主流编程语言语法支持（Rust, Python, TS, C++, TOML 等）。
- [x] **Markdown 专场**: 分屏实时预览，GitHub 风格渲染，支持图片自适应。
- [x] **多标签管理**: VS Code 风格的 Tab Bar，支持拖拽排序与未保存状态提醒。
- [x] **全局搜索**: 极速全文检索，支持文件名匹配模式。

## 2. 增强文件系统 (Advanced Workspace)
- [x] **高性能搜索**: 由 Rust 驱动的 Glob 模式文件查找 (`findFiles`)。
- [x] **实时监听**: 基于 `notify` 的文件变动监控，精准捕获增删改。
- [x] **元数据感知**: 毫秒级读取文件 `stat` 信息。

## 3. 开放插件系统 (Extensible API)
- [x] **解耦架构**: JS 插件零编译加载，完全与宿主源码隔离。
- [x] **VS Code 对标 API**: 提供 `workspace`, `system`, `window`, `views` 命名空间。
- [x] **UI 动态注入**: 支持插件自定义侧边栏 Tab 与图标。
- [x] **生命周期管理**: 完整的插件安装、启用、停用逻辑。

## 4. 智能输出控制台 (Output Console)
- [x] **按需激活**: 仅在有日志产生时，UI 才会自动显现 `Monitor` 控制台图标。
- [x] **多频道支持**: 插件可创建独立的输出频道（Channel），支持频道切换。
- [x] **独立窗口弹出**: 支持将日志面板一键弹出为独立的原生窗口（Detached Mode）。
- [x] **语法高亮**: 自动识别日志级别（Error, Warning, Success, Info）并进行染色。

## 5. 系统集成与细节 (System & Details)
- [x] **深度集成**: 一键添加/移除 Windows 右键菜单“在智码中编辑”。
- [x] **窗口恢复**: 启动时自动恢复上次关闭时的位置、大小及最大化状态。
- [x] **主题与缩放**: 深色/浅色模式动态切换，UI 字体随设置同步缩放且无抖动。
- [x] **自动更新**: 内置更新检查器，支持 GitHub/Gitee 双源下载。
