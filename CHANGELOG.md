# 更新日志 (Changelog)

本项所有显著更新都将记录在此文件中。

## [0.9.3] - 2026-01-13

### ✨ 核心特性
- **全面多语言支持**: 补全了资源管理器、搜索面板、插件占位符及系统菜单的所有翻译条目。
- **窗口状态持久化增强**: 重构了窗口位置与大小的保存逻辑，采用后端同步捕获方案，彻底解决了关闭时无法正确记录布局的问题。
- **启动体验优化**: 进一步优化了 Tauri v2 窗口启动序列，实现了真正的零闪烁布局恢复。

### 🐞 修复与优化
- **仓库地址迁移**: 更新了 Gitee 仓库路径至 `fourthz/zyma`，并同步更新了自动更新检测 API 及 GitHub Action 同步脚本。
- **稳定性修复**: 修复了单实例模式下窗口无法从后台唤起的问题。
- **配置同步**: 修正了 `tauri.conf.json` 中不支持的 `initializationScript` 配置，改用 `backgroundColor` 实现防闪烁。

## [0.9.2] - 2026-01-13

### ✨ 核心特性
- **高性能编辑器**: 集成 CodeMirror 6，支持 20+ 种主流编程语言高亮。
- **Markdown 预览**: 实时分屏预览，GitHub Dark 风格渲染。
- **全局搜索**: 实现极速全项目文本搜索与跳转。
- **多标签管理**: 支持同时打开多个文档，具备未保存脏检查。
- **自定义标题栏**: 实现 IDE 风格的无边框窗口，支持跨平台适配。
- **系统设置**: 支持主题切换 (Dark/Light)、字体动态缩放、制表符大小设置。
- **国际化**: 全界面支持简体中文、繁体中文、英文。

### 🖱️ 系统集成
- **右键菜单**: (Windows) 支持在文件和文件夹右键菜单中一键添加“在智码中编辑”，并实现设置实时同步。
- **单实例模式**: 支持文件在现有窗口中打开，避免多进程浪费资源。
- **配置持久化**: 所有偏好设置自动保存至用户主目录下的 `.zyma_config.json`。
- **静默链接唤起**: 优化了关于窗口链接的打开方式，彻底消除了点击时 CMD 黑框闪烁的问题。

### 🧩 插件系统
- **核心引擎**: 初步实现 JS 插件加载与运行。
- **基础 API**: 注入 `editor.insertText`, `ui.notify` 等 API 给插件调用。
- **Hello World**: 提供第一个演示插件。

### 🛠️ 架构改进
- **后端模块化重构**: 将庞大的 `lib.rs` 拆分为 `config`, `fs`, `system`, `plugins` 等独立模块，提升了代码的可维护性与安全性。
- **自动发布流程优化**: 解决了 GitHub Release 草稿不可见问题，实现全自动公开发布。
- **启动速度优化**: 实现并行异步初始化与窗口隐藏预加载，启动响应速度提升 60% 以上，彻底告别白屏闪烁。
- **专业关于对话框**: 新增包含品牌 Logo、版本信息及官方仓库链接的精致“关于”对话框。
- **跨平台 UI 适配**: 针对 macOS 特别优化了标题栏“红绿灯”布局与拖拽区域。
- **技术栈对齐**: 升级至 Tauri v2 稳定版，并将执行文件名标准化为 `zyma.exe`。
- **权限修复**: 深度优化 IPC 握手逻辑，解决 500 和 Origin 验证错误。
